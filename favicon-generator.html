Favicon Generator Guide

This guide will help you create new favicons for your website based on your updated logo.

Step 1: Use an Online Favicon Generator

Since we don't have ImageMagick installed, you can use one of these online favicon generators:

  • Real Favicon Generator (Recommended)
  • Favicon.io
  • Favicon & App Icon Generator

Upload your logo file (assets/images/logo.png) to one of these services.

Step 2: Configure Your Favicon Settings

When using Real Favicon Generator (recommended):

  1. Upload your logo.png file
  2. Adjust settings for each platform (iOS, Android, Windows, etc.)
  3. For the SVG favicon (Safari pinned tab), choose an appropriate silhouette color
  4. Generate your favicon package
  5. Download the generated package

Step 3: Replace Existing Favicon Files

Extract the downloaded package and replace the following files in your assets/images/ directory:

- favicon-16x16.png - favicon-32x32.png - apple-touch-icon.png - android-chrome-192x192.png - android-chrome-512x512.png - safari-pinned-tab.svg - site.webmanifest (if provided)

Note: If the generator provides a different set of files or different naming convention, you may need to update the references in _includes/head/custom.html.

Step 4: Update site.webmanifest (if needed)

If the generator provides a new site.webmanifest file, make sure to update it with your site's information:

{
    "name": "DO-178C Development Assurance",
    "short_name": "DO-178C",
    "icons": [
        {
            "src": "/assets/images/android-chrome-192x192.png",
            "sizes": "192x192",
            "type": "image/png"
        },
        {
            "src": "/assets/images/android-chrome-512x512.png",
            "sizes": "512x512",
            "type": "image/png"
        }
    ],
    "theme_color": "#ffffff",
    "background_color": "#ffffff",
    "display": "standalone"
}

Step 5: Test Your New Favicons

After replacing the files:

  1. Run your Jekyll site locally or deploy it
  2. Clear your browser cache
  3. Visit your site and verify the new favicons appear correctly
  4. Test on different browsers and devices if possible

Pro Tip: For the best results, start with a high-quality, simple logo that works well at small sizes. Avoid complex details that won't be visible at favicon sizes.
